## YouTube Fetch Limitation

All four YouTube URLs returned 302 redirects to `consent.youtube.com` — the EU/Swedish cookie consent wall blocks server-side access. Video metadata is available only from:
1. Existing indexes (youtube_index.md from wandering-anchor-book)
2. Master prompt video timestamps (Home@ix Concept Explained, Docklands Years)
3. COPE Brain node names (which reference some videos by title)
4. Local audio/video files on disk

A full channel scrape would require `yt-dlp --flat-playlist` run locally, or YouTube Data API v3 with an API key.
